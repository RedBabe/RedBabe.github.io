# DUE

> 非专业的符号语言

## 版本

0.0.1

## API

| 符号 | 解释 |
|-|-|
| `/` | 好的，嗯嗯 |
| `#` | 不懂，懵 |
| `^` | 笑，开心，期待 |
| `!` | 激动，吃惊 |
| `~` | 很棒，喜欢 |
| `=` | 等等 |
| `...` | 无语 |
| `@@` | 晕 |
| `<` | 少点，轻点，温柔点，表意图 |
| `>` | 多点，刚点，霸道点，表意图 |
| `1` | 是，有，表肯定 |
| `0` | 不是，没有，表否定 |
| `+` | 较多，较好，较正向，表状态 |
| `-` | 较少，较差，较反向，表状态 |
| `,` / `，` | 让我想想，思考一下 |
| `.` / `。` | 话题结束，不聊这个 |

## 说明

符号重复数量表程度，共3级。如 # ## ###
